The Blog Portlet allows select individuals to author blogs that can be read and commented upon by the entire campus population.

Viewing Blogs

As a blog reader there are a number of options for viewing information. By default users will be presented with a list of blog entries organized by date within the top-level category of the blog.

To view additional blog entries:

  1. Click a category name under the Subcategories heading to view blog entries for another subject area.
  2. To return to the parent category view, click the parent category name listed above the blog entries.
  3. To view comments for a blog entry, click the Comments link.
  4. To view Trackbacks for a blog entry, click the Trackback link.

To Search for blog entries:

  1. Enter word to be search for.
  2. Click the Search icon.
  3. Blog entries for all categories matching your search criteria will be displayed. Entries will be sorted by date

To review blog entries for a specific date:

  1. Navigate to the desired month, using the forward and back icons within the Blog calendar
  2. Dates for which blog entries exist will be highlighted in the calendar.
  3. Click a highlighted calendar day to view all blog entries submitted on that date. Entries from all categories and sub-categories will be included in the display.

 

 

Adding Blog Comments

If comments have been enabled for a blog, then any blog reader can add a comment to a specific blog post.

To add a comment:

  1. Click Comments link next to the blog entry you wish to comment on.
  2. Existing comments will be displayed along with fields to enter new comments.
  3. Enter your name and comment. E-mail address and URL may also be entered, but this information is optional.
  4. Click the Comment button.

Blog Administration

Individuals with privileges to author blog content, maintain blog properties or settings will be given access to the Blog administration console. From the administration console, you will be able to access features for managing blog properties, adding categories, creating and editing blog entries and uploading files.


Setting Blog Properties

To manage the settings of your blog:

  1. Click the Properties Tab within the Blog Portlet Administrative Console.
  2. Current blog settings are displayed.
  3. Update the properties you wish to edit. A complete listing of the properties is provided below.
  4. Click the Edit blog properties button.
  5. To return to the original settings, prior to clicking the Edit blog properties button, click the Reset button.

Managing Categories

Blog categories define the structure and organization for blog entries and make finding information easier for end users.

To add a blog category:

  1. Click the Categories Tab within the Blog Portlet Administrative Console.
  2. Enter the Category name.
  3. Enter Category description. (optional)
  4. Enter Category meta-data. (optional)
  5. Select a parent category from the drop down. If no parent is selected, the new category will be placed at the root level of the blog.
  6. Click the Add category button.

 

To edit a blog category:

  1. Click the Categories Tab within the Blog Portlet Administrative Console.
  2. Select the category to edit from the category name drop down under the Modify an existing blog category section.
  3. Click the Edit category button.
  4. Update Category description. (optional)
  5. Update Category meta-data. (optional)
  6. Click the Update category button.

To delete a blog category:

  1. Click the Categories Tab within the Blog Portlet Administrative Console.
  2. Select the category to delete from the category name drop down under the Delete an existing blog category section.
  3. Click the Delete category button.

 

Managing Blog Entries

Adding Entries

Blog Entries are the heart of the blog - delivering the information users want to read. Every blog entry must be organized within a category.

To add a blog entry:

  1. Click the Add Entry Tab within the Blog Portlet Administrative Console.
  2. Enter the Blog Entry title.
  3. Enter the Blog entry text. You can include formatting by inserting quick tags within the entry field.
  4. Select a category from the category name drop down.
  5. Enter Blog trackback URLs. (optional)
  6. Enter Blog ping URLs. (optional)
  7. Select option to enable or disable comments.
  8. Select option to enable or disable trackbacks
  9. Specific the date and time the entry should be published. Use the format MM/dd/yyyy HH:mm:ss (optional)
  10. Enter a proposed name. (optional).
  11. Click the Add blog entry button.

 

Editing Entries

To edit a blog entry:

  1. Click the Edit Entries Tab within the Blog Portlet Administrative Console.
  2. Select the category to which the entry has been associated in the Blog Category drop down under the Edit entries in an existing blog category section.
  3. Click the Edit entries in an existing blog category button.
  4. Select the item to be edited from the Blog entry drop down under the Edit an entry from ... section.
  5. Click the Edit blog entry button.
  6. Make the necessary changes.
  7. Click the Update blog entry button.


Deleting Entries

To delete a blog entry:

  1. Click the Edit Entries Tab within the Blog Portlet Administrative Console.
  2. Select the category to which the entry has been associated in the Blog Category drop down under the Edit entries in an existing blog category section.
  3. Click the Edit entries in an existing blog category button.
  4. Select the item to be deleted from the Blog entry drop down under the Delete an entry from ... section.
  5. Click the Delete blog entry button. Blog entry is immediately deleted, no additional confirmation message will be displayed.
