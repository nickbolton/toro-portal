Managing Blog Entries

Adding Entries

Blog Entries are the heart of the blog - delivering the information users want to read. Every blog entry must be organized within a category.

To add a blog entry:

  1. Click the Add Entry Tab within the Blog Portlet Administrative Console.
  2. Enter the Blog Entry title.
  3. Enter the Blog entry text. You can include formatting by inserting quick tags within the entry field.
  4. Select a category from the category name drop down.
  5. Enter Blog trackback URLs. (optional)
  6. Enter Blog ping URLs. (optional)
  7. Select option to enable or disable comments.
  8. Select option to enable or disable trackbacks
  9. Specific the date and time the entry should be published. Use the format MM/dd/yyyy HH:mm:ss (optional)
  10. Enter a proposed name. (optional).
  11. Click the Add blog entry button.

 

Editing Entries

To edit a blog entry:

  1. Click the Edit Entries Tab within the Blog Portlet Administrative Console.
  2. Select the category to which the entry has been associated in the Blog Category drop down under the Edit entries in an existing blog category section.
  3. Click the Edit entries in an existing blog category button.
  4. Select the item to be edited from the Blog entry drop down under the Edit an entry from ... section.
  5. Click the Edit blog entry button.
  6. Make the necessary changes.
  7. Click the Update blog entry button.


Deleting Entries

To delete a blog entry:

  1. Click the Edit Entries Tab within the Blog Portlet Administrative Console.
  2. Select the category to which the entry has been associated in the Blog Category drop down under the Edit entries in an existing blog category section.
  3. Click the Edit entries in an existing blog category button.
  4. Select the item to be deleted from the Blog entry drop down under the Delete an entry from ... section.
  5. Click the Delete blog entry button. Blog entry is immediately deleted, no additional confirmation message will be displayed.
