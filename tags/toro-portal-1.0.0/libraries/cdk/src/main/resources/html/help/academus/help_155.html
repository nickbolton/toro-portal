Submitting Announcements:Communicate with Current Enrollees

 

About Announcements

By adding announcements, you can communicate with everyone in your offering, as well as refer to these communications in the future. These announcements are only visible to users who are enrolled in the current offering.


Adding an Announcement

To add an announcement to an offering, locate the Announcements channel and follow these steps:

  1. Click on the Add icon at the top of the Announcements channel.
  2. Type your message in the provided text area.
  3. Click the Spell Check link to check the contents of your message.
  4. When you are finished typing, click the Submit button.
  5. All users will now see this announcement, as well as the date it was created.
