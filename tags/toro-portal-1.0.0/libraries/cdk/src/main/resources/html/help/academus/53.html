Editing My Profile

To change your email address:

  1. Locate the User Administration channel and click the Edit my Profile link at the top of the channel.
  2. You should now see the Edit My Profile view, which allows you to select one of two options.
  3. Select the Edit My Email Address link.
  4. Enter your current password for verification purposes.
  5. Revise your email address.
  6. Click the Submit button.

 

 

To change your password:

  1. Locate the User Administration channel and click the Edit my Profile link at the top of the channel.
  2. You should now see the Edit My Profile view, which allows you to select one of two options.
  3. Select the Edit My Password link.
  4. Enter your current password.
  5. Enter your new password.
  6. Enter your new password again for verification.
  7. Click the Submit button.
