Searching for Groupware Users : Searching for Users

 

To search for a user who has already been added to the system, follow these steps:

  1. Locate the User Administration channel and click the Search Users option.
  2. You will now see the Search for User screen.
  3. Enter as little or as much information as you know about the user in any or all of the four fields provided (User Name, First Name, Last Name, Email).
  4. Click the Submit button.
  5. The system will return all matches for your criteria. If no users are found, click the New Search button, make your search less specific, and try again until you get a satisfactory result.
