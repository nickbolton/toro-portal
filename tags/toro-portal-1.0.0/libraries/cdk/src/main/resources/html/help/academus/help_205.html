Import Channel : Importing Users or Offerings

 

The Import Channel allows you to import new users and offerings into the system. User import files support the following data:

  • Username
  • First name
  • Last name
  • Email address
  • User role

Offering import files support the following data:

  • Offering name
  • Description
  • User enrollment model
  • Default type for enrolled users

Any errors encountered during the import will be displayed within the channel

 - Allows you to choose Users or Offerings for import.

Importing a roster for an offering is addressed within the Roster Channel.
