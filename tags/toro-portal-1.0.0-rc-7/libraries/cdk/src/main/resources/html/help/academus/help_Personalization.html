To begin personalizing UNICON Academus, locate and click the Customize link in the upper-right corner of the portal window. For some Academus deployments the feature may be labeled Preferences.

 

 

There are several links at the top of the page when you are in customization mode.

  • Change Content & Layout- This feature allows you to add or remove tabs, columns, and channels from your layout. You can also rearrange or rename any non-restricted tabs, columns, and channels in your layout.
  • Pick Colors - This will change the colors, and style of UNICON Academus. The styles available for selection will vary by institution.
  • Manage Fragments - This option will be available for a limited number of users granted privileges to manage fragments within the portal.
  • Change Languages - Use this link to change the language in your view.

 

 

Setting Up Your Personal Tabs

An individual's personal tabs are places where each user can set up and group channels for their own use. The user is then responsible for the layout and channels contained within their new tab.

Before you can begin grouping and placing channels in a tab, you need to create the tab. Start by clicking on the Customize link at the top of the screen to enter Preferences Mode. This link may be identified as Preferences for some institutions. Then follow these instructions:


Creating a New Tab
  1. From the Change Content & Layout view, click the Add a new tab link near the top of the window.
  2. A new tab titled Unnamed will be placed in the right most position. This tab will be highlighted.
  3. Click the Rename Tab link.
  4. Enter the name for the new tab.
  5. Click Rename.
  6. Clicking on the Left Arrow icon next to the tab name will allow you to position the tab in the desired location.
  7. Click Save Settings to save your tab and exit preferences model, or proceed to add columns and content.

 

 

 

Moving, Renaming or Removing a Tab

To Move a Tab:

  1. First, you must select the tab to be moved. Click on the tab you want to move.
  2. You should notice that the tab's color has changed, and arrow icons are enabled on each side of the tab name.
  3. To move the tab one position to the left, click the Left Arrow icon.
  4. To move the tab one position to the right, click the Right Arrow icon.

(You may have to click the Left or Right Arrowsseveral times to position the tab exactly where desired.)

To Rename a Tab:

  1. First, you must select the tab to be renamed. Click on the tab you want to rename.
  2. You should notice that the tab's color has changed.
  3. Click the Rename Tab link.
  4. Enter the new tab name.
  5. Click Rename.

To Remove a Tab:

  1. First, you must select the tab to be removed. Click on the tab you want to remove.
  2. You should notice that the tab's color has changed.
  3. Click the Delete Tab link.
  4. The system will verify that you really want to remove the tab. Click OK to confirm.
  5. The tab and all of its contents have now been removed.

To save any of these changes and exit preferences mode, click Save Settings.

Columns and Channels

After creating a new tab, the next step is populating and customizing it with columns and channels. A new tab will contain one column for content by default. To add an additional column, follow these steps:

  1. First, you must select the tab where you want to add the column.
  2. Click the Add New Column link to add a new column.
  3. Next select the desired column sizing by click the desired column presentation option. In a theme which includes a static first column, two options are provided for the other two columns either evenly sized or one wide, one narrow column.
  4. You can remove a column at any time by clicking the Remove Column link. You will be prompted to confirm the deletion prior to the removal of the column.

Once you have a column in place, you can add channels or portlets to the tab. To Subscribe to a channel, follow these steps:

  1. First, you must select the tab where you want to add channels or portlets.
  2. Click the Add Content link.
  3. Browse or Search through the channel listing to locate the channel you would like to add. When browsing, clicking the category name will display the list of available channels and portlets within that category.
  4. Once the desired channel has been located, click on the channel or portlet name, to view a description and access the subscribe option.
  5. Click the Subscribe to this channel link to add that channel to your layout.
  6. The channel will be placed in the first column in the top position. Use the arrow icons displayed within the channel to move the channel to the desired position.
  7. To remove an existing channel, click the Remove button in the upper right hand corner of the channel.

To save new columns and content layouts and exit preferences mode, click Save Settings.

 

 

 

Restricted Tabs

Certain tabs called fragments are created and managed by portal administrators. These tabs are not modifiable by an end user during the customization process. These tabs will be identified as locked by the system when selected in the Change Content and Layout view.

 

 

 

Choose a Skin

What is a Skin?

Skins are pre-produced visual schemes that can be applied to the user interface portion of a program. UNICON Academus makes use of skins to change the colors, fonts, graphics, and general layout of the application to provide the end user with several appealing choices. When someone changes skins on their layout, it only affects them. This does not affect people who log in through other user accounts.

Picking Colors

To view and select from a list of available skins, simply follow these steps:

  1. Click the Customize link in the upper right corner of the portal.
  2. Click the Pick Colors link.
  3. The application displays a list of available skins, as well as thumbnail representations for each. Each skin is accompanied by its name and a short description.
  4. Select a skin by clicking the radio button to the left of its thumbnail image.
  5. Click the Change Skin button.
  6. Click the Save Settings button to save all preference changes and exit Customizing your Portal.
  7. The new skin will now be applied to all views of Academus.
