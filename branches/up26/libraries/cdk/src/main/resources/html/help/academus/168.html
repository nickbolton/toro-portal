Export an Offering

To export an offering into a delimited file, simply follow these steps:

  1. Locate the Offering Administration channel.
  2. Find the Offering you would like to export using the search function, and click the Export icon next to it.
  3. You will be prompted to save this file to your local drive. Select a place to save it and click OK.

The exported file is a ZIP compressed file which contains a file called "offeringExport.xml." This file contains information from the Offering that is not specific to a single user. The exported file is in XML format, and it contains the following information:

  • Offering ID
  • Offering Name
  • Offering Description
  • Offering Start and End Dates
  • Offering Meeting Days and Times
  • Offering Location
  • Enrollment Model
  • Roster Channel enrollment data
