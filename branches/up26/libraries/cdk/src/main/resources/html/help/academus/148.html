Adding an Event to an Offering Calendar

To add a calendar event, simply click on any of the Add icon buttons on the Offering Calendar channel. If these buttons do not appear, then the logged in user does not have permissions to add events.

The Add New Event screen consists of three parts:

  • New Event - Input the event details here.
  • Event Attendees - Send notifications to all users enrolled in the offering regarding the event.
  • Event Recurrence - This section makes it easy to repeat this event many times in the future, based upon a regular schedule.

 

New Event Section

  1. In the Add Event section, first give this event a name in the Title text field.
  2. Make sure the Date fields are correct (month, day, and year).
  3. Choose a Time, or select All-Day if this is event will take place all day.
  4. Enter the Place where the event will occur.
  5. For the Category field, there are many choices (i.e. Appointment, Meeting, Holiday, etc.). Select the one that best applies to this event.
  6. For the Priority field, there are several choices, ranging from Very Low to Very High. These priorities each come with a color-coded box that appears next to each event on the calendar date views.
  7. If there are any extra notes about this event, enter them on the Notes text area at the bottom left of the channel. You have the option to Spell Check content entered in the notes section.


Event Attendees Section

A notification can be sent to users enrolled in the offering in the form of an email or a message in the Notifications channel. Simply check the box(es) for the desired type of notification.


Event Recurrence Section

The next thing that needs to be determined is the Recurrence Pattern of this event (if any).

  1. If this event is a one-time event only, leave the None button selected.
  2. If this event will be a daily event, select the Every X day(s) button, and make sure the number 1 is entered in the text field. If the event occurs once every three days, enter the number 3 in the text box, etc.
  3. If this event is a weekly event, select the Every X week(s) button, and make sure the number 1 is entered in the text field. If the event occurs once every two weeks, enter the number 2 in the text box, etc.
  4. If you selected the Every X week(s) option, select a day or days of the week upon which this event falls. You can select more than one day for this field.
  5. If this event is a monthly event, select the Every X month(s) button, and make sure the number 1 is entered in the text field. The event will be scheduled on the same day of each month. If the event occurs every other month, enter the number 2 in the text box.
  6. If this is an annual event, such as a birthday or anniversary, select the Every year field.
  7. Now that the recurrence pattern has been established, it is time to decide the Range of Recurrence, or in other words, how many times will this event take place?
  8. If there is a specific number of occurrences, select the End After: X occurrences section and enter that number in the text field.
  9. If there is an end date to this recurrence, select the End By: section and choose a date from the drop-down menus.
