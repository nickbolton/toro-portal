View All Members in an Offering

The roster channel appears whenever you are viewing a created offering. The default view will show the users who are currently enrolled in the offering. This includes the Sponsor (or instructor) of the offering.

The roster list consists of four pieces of information about each user:

  • User ID
  • Name
  • Role in the offering
  • Enrollment Status

View a Roster Member's Information

If you would like to see more information on any specific user who is enrolled in this offering, simply click on their name or click on the magnifying glass icon next to their name in the main roster view.

If available, you should see the following information:

  • User ID
  • User's First and Last name, as well as Middle Initial
  • User Type
  • User Status
  • Email Address
