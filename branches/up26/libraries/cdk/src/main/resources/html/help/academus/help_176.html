Managing System Notifications

Select system users will have access to manage system messages. These administrative users will have the ability to view any message in the system, regardless of author. The ability to delete and/or export any message is also provided.

To delete messages from the system:

  1. Click the System Messages tab within the Notification Portlet.
  2. Locate the message(s) to be deleted. Messages are sorted by Received Date by default. To page through additional messages, use the Prev and Next links at the bottom of the channel.
  3. To view the contents of a message prior to deleting, click the message subject.
  4. Click the checkbox associated with all messages to be deleted.
  5. Click the Delete button.
  6. You will be presented with a list of the selected messages and asked to confirm the deletion. Click the Yes radio button.
  7. Click Submit to complete the delete process.
