Edit or Delete an Offering : Modify Offerings

 

To Edit an Offering, complete these steps:

  1. Locate the Offering Administration channel.
  2. Find the Name of the Offering you would like to edit via the search function, and click the Edit icon next to it.
  3. You will be able to edit all of the offering details including Topic Name, Offering Name, Offering Description, User Enrollment Model, and Enrolled User Default Type.
  4. Click the Submit button when all of your edits are finished.

 

To Delete an Offering, complete these steps:

  1. Click the Delete icon next to the Offering you'd like to delete. This will not delete the Offering permanently, but rather, it will change its status to Inactive.
  2. Select the Yes radio button and click Submit to confirm that you intend to inactivate this Offering.
  3. To permanently Delete this Offering, you must first be able to View Inactive Offerings. Click the Inactive link at the top of the channel.
  4. You will now see all Inactive Offerings. To permanently Delete an offering from this view, click the Delete icon next to that Offering.
  5. In the confirmation screen, select the Yes radio button and click Submit.
  6. The Offering will now be permanently deleted from the system.
