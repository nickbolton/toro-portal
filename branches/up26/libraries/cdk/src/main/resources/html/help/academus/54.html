The Gradebook Channel:Using the Gradebook

The Gradebook Channel is an interface for scores, grades, exam results, and assignments associated with the current offering. As an instructor, you can view all enrolled members' scores for each offering. As a member, you can view only your scores for a particular offering.

From the main view, users can:

View Scores [default] - shows a listing of users associated with the current offering cross-indexed with exams and/or assignments. Where applicable, a grade is displayed for each user for the exam or assignment. User names are listed down the left side, and total grades down the right side. In between are displayed columns, each column representing an exam or assignment. Underneath the user rows are two rows that display the calculated mean and median of the columns.

- brings up an interface in the user's operating system for exporting a file to a local destination. The file contains all of the gradebook data, which can be imported into other applications.

- displays the view for current and pending exam and assignment activations.

- displays the view for the adding of a new gradebook column (exam or assignment) to the current offering.

- displays the view for editing the weighting of the current exams and assignments in the gradebook.

From the main view, users can select a gradebook column to manage:

- displays all exams or assignments submitted for the selected column.

- displays the view for activating the selected exam or assignment. Once activated, the active exam or assignment will be displayed in the Activations main view.

- displays a view for entering or editing the scores of the selected column. This is a quick way to enter exam (or assignment) scores for all offering users for the selected column.

- displays the view for editing the properties of the selected column.

- displays the view to confirm or cancel the deletion of the selected column from the gradebook.
