Import Users or Offerings : Quick Population Method

 

Why Import?

Importing Users and Offerings is a quick way to get the system populated with your information. It is much quicker than manually creating every user and offering, one at a time. When you import a file of users, for example, you could list 100 users in 100 lines. The import time would take a couple seconds. Creating these user accounts manually could take hours.


Preparing the User Import File

The file you import must be a text file, and it must contain the following information in this order:

  • User Name
  • First Name
  • Last Name
  • Email Address
  • User Role

The file itself must contain XML tags in order for the system to read them in correctly. Here's the correct format for a file:

<pre>
<userList>
<user>
<username>bob73</username>
<firstname>Bob</firstname>
<lastname>Jones</lastname>
<email>bob@university.edu</email>
<role>User</role>
</user>
<user>
<username>soozy</username>
<firstname>Sue</firstname>
<lastname>Zagg</lastname>
<email>sue@university.edu</email>
<role>User</role>
</user>
</userList>
</pre>

This file would create two users:

  • "bob73" -- Bob Jones, bob@university.edu, with a type of User
  • "soozy" -- Sue Zagg, sue@university.edu, with a type of User

Preparing the Offering Import File

The file you import must be a text file, and it must contain the following information in this order:

  • Offering Name
  • Offering Description
  • User Enrollment Model
  • Enrolled User Default Permissions

The file itself must contain XML tags in order for the system to read them in correctly. Here's the correct format for a file:

<offeringList>
<offering>
<name>ENG101</name>
<description>English101</description>
<enrollmentModel>Open</enrollmentModel>
<role>Member</role>
</offering>
<offering>
<name>PSY101</name>
<description>Psychology101</description>
<enrollmentModel>Open</enrollmentModel>
<role>Member</role>
</offering>
</offeringList>

This file would create two offerings:

  • ENG101 - English 101, Open Enrollment, Members by default.
  • PSY101 - Psychology 101, Open Enrollment, Members by default.

Importing Users or Offerings

Now that you have the correct file format, importing the file itself is easy.

  1. In the Import channel, select Users or Offerings from the Import Type drop down.
  2. Click the import icon next to the Import Type drop down.
  3. If you know the path to the file, type it in the text field. Otherwise, click the Browse button to locate it on your computer.
  4. Once you've found the file, click the Submit button to import the users or offerings.
  5. You will get a confirmation screen showing the users or offerings successfully imported.
