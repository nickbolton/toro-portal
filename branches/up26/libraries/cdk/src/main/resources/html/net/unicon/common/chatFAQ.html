ACADEMUS™ Chat FAQ

Why do I get just an empty window when trying to launch the chat client?

The chat client requires that the browser used to access ACADEMUS™ has a version of the Java™ Runtime Environment (JRE) installed.


How does one obtain a version of the JRE?

Download and install a version of the Java™ Runtime Environment (JRE) from Sun Microsystems.

NOTE: The chat client has been tested and found to work with the following JRE versions: 1.3.1, 1.4.2, and 1.5.0. Other versions may also work but if trouble persists, please try installing one of these versions.
