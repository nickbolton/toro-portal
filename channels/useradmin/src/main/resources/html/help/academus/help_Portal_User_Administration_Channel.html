The User Administration channel allows the user, if they have appropriate system permissions, to add and edit users within the system. This channel is primarily used to support Academus implementations that do not use an extended authentication method, such as LDAP or Active Directory.

 Allows the user to search for other users within the system. Supported search criteria includes username, first name, last name, and/or email address.

 Allows the user to add new users to the system

 Allows the user edit their personal profile including updates to password and email address.

Adding Portal Users

To add a new user to the system, complete the following steps:

  1. Locate the User Administration channel and click the Add User link at the top of the channel.
  2. This will take you to the Add User view where you must provide information about the new user.
  3. Type a User Name in the first text field. The User Name must be unique to the system. If anyone else is already using the User Name you provide, you will have to choose another one.
  4. Now select a Password. If you want the Password to be the same as the User Name, then select the Automatic Password radio button. Otherwise, select the Manually Entered radio button and type a Password in the text field provided.
  5. Type the user's First Name in the First Name field.
  6. Type the user's Last Name in the Last Name field.
  7. Type the user's Email Address in the Email field.
  8. Once you are satisfied with all of your selections, click the Next button to add this new user.

The next view will allow you to select the groups in which this new user will be enrolled.

  1. Select groups by clicking the check box next to the desired group(s) and click Submit.

 

 

Searching for Portal Users

To search for any users who have already been added to the system, follow these steps:

  1. Locate the User Administration channel and click the Search Users link at the top of the channel.
  2. You should now see the Search for User screen.
  3. Enter as little or as much information as you know about the user in any or all of the four fields provided (User Name, First Name, Last Name, Email).
  4. Click the Submit button.
  5. The system will return all matches for your criteria. If no users are found, make your search less specific and click the Search button at the bottom of the channel. Continue to revise search criteria until you get a satisfactory result.

 

 

Editing My Profile

To change your email address:

  1. Locate the User Administration channel and click the Edit my Profile link at the top of the channel.
  2. You should now see the Edit My Profile view, which allows you to select one of two options.
  3. Select the Edit My Email Address link.
  4. Enter your current password for verification purposes.
  5. Revise your email address.
  6. Click the Submit button.

 

 

To change your password:

  1. Locate the User Administration channel and click the Edit my Profile link at the top of the channel.
  2. You should now see the Edit My Profile view, which allows you to select one of two options.
  3. Select the Edit My Password link.
  4. Enter your current password.
  5. Enter your new password.
  6. Enter your new password again for verification.
  7. Click the Submit button.
