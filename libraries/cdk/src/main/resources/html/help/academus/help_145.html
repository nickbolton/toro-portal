Calendar Search:Locating Events

 

To search for scheduled events, simply locate the Calendar channel and click the Search link located at the top of the channel. The following Search screen will be displayed:

To perform a search, simply follow these steps:

  1. Enter the search text in the Find field.
  2. Now select which field of the Event that this search should take place: Title, Place, and/or Notes. You can search in more than one place.
  3. Choose a Category for your search (if desired).
  4. To refine the search to a specified time span, enter the dates in the From and To fields.
  5. The Calendar field will default to the title of this offering. No other selection options will be provided in this view.
  6. Click the Search button to begin the search.
